# Supermarket-Sales-Analysis
This project analyzes supermarket sales data using SQL and Power BI to identify sales trends, customer behavior, and key revenue-driving factors.
## Project Objectives

- Analyze sales performance across product categories
- Identify customer purchasing behavior
- Understand time-based sales trends
- Generate business insights using SQL
- Build an interactive Power BI dashboard

- ## Tools & Technologies

- SQL
- Power BI
- Excel

- ## Dataset Overview

- Total Transactions: 1000
- Time Period: January – March
- Features: Product Line, Sales, Quantity, Customer Type, Payment Method, Rating, Time

- ## Key Insights

- Food & Beverages generated the highest revenue
- Member customers contributed more sales
- Sales peaked during morning hours
- Customers purchased an average of 5.5 items per transaction
